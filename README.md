# Потолок и Точка — натяжные потолки в Москве

Два лендинга-визуализации бренда «Потолок и Точка». Натяжные потолки в Москве и МО с финальной ценой, AI-визуализацией по фото комнаты и личным контактом владельца.

**Сайт:** https://premium-ceiling-landing.vercel.app
**A/B вариант:** https://premium-ceiling-landing.vercel.app/v2
**Документация бренда:** https://github.com/dimas9491/potolok-i-tochka-docs (приватный)

## Главное отличие сайта от рынка

**«Цена в рекламе = цена в чеке»** — мы продаём готовыми комплектами с финальной ценой за комнату. Никаких «от 290 ₽/м²» с доплатой на замере. Главное УТП — **AI-визуализация по фото комнаты бесплатно за 1-2 дня до замера**.
